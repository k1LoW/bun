# [PoC] Bun [![Build Status](https://travis-ci.org/k1LoW/bun.svg?branch=master)](https://travis-ci.org/k1LoW/bun)

Lazy Japanese paragraph extractor.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'bun'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bun
